EN

Installing Simple_WebRadio_flash_0x0.bin

If you want to do this without the Arduino IDE, the binary file is available.
You can also flash this using the following applications:

- Flash Download Tool - https://docs.espressif.com/projects/esp-test-tools/en/latest/esp32c6/production_stage/tools/flash_download_tool.html
- ESPConnect - https://thelastoutpostworkshop.github.io/ESPConnect/

Photos of the settings are included in the folder!
